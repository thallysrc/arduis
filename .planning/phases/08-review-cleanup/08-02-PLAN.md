```python
def run_git_async(argv: list[str], on_done, runner: HostRunner | None = None) -> None:
    wrapped = (runner or HostRunner()).wrap_argv(argv)        # route through the seam
    proc = Gio.Subprocess.new(
        wrapped,
        Gio.SubprocessFlags.STDOUT_PIPE | Gio.SubprocessFlags.STDERR_PIPE,
    )
    def _cb(p, res):
        ok, out, err = p.communicate_utf8_finish(res)
        on_done(p.get_exit_status(), out or "", err or "")
    proc.communicate_utf8_async(None, None, _cb)
```

  ```python
      flags = Gio.SubprocessFlags.STDOUT_PIPE | Gio.SubprocessFlags.STDERR_PIPE
  ```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| branch name → gh argv | the branch is user-chosen and crosses into a gh invocation as a discrete argv element. |
| gh `--json` output → display only | PR JSON is `json.loads`'d and DISPLAYED (subline); it is NEVER used to build a path or command. |
| worktree dir → run_git_async cwd= | the cwd is a TRUSTED internal path arduis constructed (a worktree dir), never raw user input. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-04 | Tampering / EoP | branch injected into a gh argv | mitigate | gh argv is a Python LIST with the branch a DISCRETE element (mirrors git_service T-02-01); no shell, no `shell=True`; routed through HostRunner by the Wave-2 caller. |
| T-08-05 | Injection | gh JSON output treated as code/path | mitigate | `parse_pr_view` does `json.loads` + a dict shape-guard and the result is DISPLAYED only (subline) — never eval'd, never used to build a path/command. |
| T-08-06 | DoS (self) | gh rate-limit storm | mitigate | gh is read behind the Plan-01 TTL cache (GH_TTL_S=120) + a Wave-2 in-flight debounce; gh_available() short-circuits when gh is absent (no call); exit-4 is a STATIC degrade state (no retry). |
| T-08-07 | Tampering (the ONE write) | `gh pr create` is the only sanctioned write | accept | opening a PR is roadmap-sanctioned (REVIEW-02); the `--web` form keeps it browser-driven and read-only-in-app afterward. The builder is asserted by argv shape only and NEVER executed in tests. |
</threat_model>
