## Terminais default da task

| Option | Description | Selected |
|--------|-------------|----------|
| Agente + shell por repo | 2 repos = 4 terminais; espelha o hábito tmux (múltiplos claude em paralelo); 1 repo = default 03.1 | ✓ |

**User's choice:** Agente + shell por repo (recomendado)
