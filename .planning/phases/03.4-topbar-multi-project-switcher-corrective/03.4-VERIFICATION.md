### G-1 Gap (UAT-Reported, Resolved)

The UAT found that opening a single-repo project via "Abrir projeto" showed a blank canvas (no terminal).

**Code verification:**

1. `_switch_project` seeds the main scratch shell on first activation (line 2538):
   ```
   if active is not None and f"{_MAIN_SID}:t0" not in self._leaf_by_sid:
       self._open_shell_leaf()
   ```
   `_open_shell_leaf` is called inside `_switch_project`. CONFIRMED.

2. `_resolve_members` adds the degenerate single-repo fallback (line 2334): if `detect_member_repos` returns `[]`, it checks `git rev-parse --is-inside-work-tree` and returns `[basename(root)]` if the root is a git repo. CONFIRMED.

Regression test `test_switch_to_unseeded_project_seeds_its_main_shell` in `test_window_projects.py` PASSES. Commit `4a93988` (referenced in HUMAN-UAT.md).
