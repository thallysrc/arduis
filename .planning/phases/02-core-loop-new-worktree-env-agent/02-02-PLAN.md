`Adw.TabView`/`TabBar` whose `+` button opens the "New worktree" dialog, creates the worktree

`cwd`, and generalizes the Phase-1 no-orphan close path to tear down ALL active sessions.

gi-importing presentation module and the logic in the tested domain layer.

checklist for the GUI/teardown behaviors that cannot be unit-tested under Wayland.

    ```python
    from __future__ import annotations
    import gi
    gi.require_version("Gtk", "4.0")
    from gi.repository import Gio  # noqa: E402
    from arduis.host_runner import HostRunner

    def run_git_async(argv: list[str], on_done, runner: HostRunner | None = None) -> None:
        """Run a git argv off the GTK loop; on_done(exit_status, stdout, stderr) fires on the main loop."""
        wrapped = (runner or HostRunner()).wrap_argv(argv)   # route through the seam
        proc = Gio.Subprocess.new(
            wrapped,
            Gio.SubprocessFlags.STDOUT_PIPE | Gio.SubprocessFlags.STDERR_PIPE,
        )
        def _cb(p, res):
            ok, out, err = p.communicate_utf8_finish(res)
            on_done(p.get_exit_status(), out or "", err or "")
        proc.communicate_utf8_async(None, None, _cb)
    ```
    Do NOT introduce threading/asyncio (CLAUDE.md footgun). The on_done callback runs on the GLib main loop, so it is safe to mutate widgets/SessionStore there.
