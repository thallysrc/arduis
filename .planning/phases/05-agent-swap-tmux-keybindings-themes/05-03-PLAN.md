## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user config (agent/keys/theme) → window runtime | config values drive the fed command, key dispatch, and palette |
| set_theme action target → palette selection | a GVariant string slug selects a theme |
| focused terminal ← refeed bytes | the configured agent command is written into a live PTY |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01 | Tampering/EoP | configured agent command fed into the shell | mitigate | The bytes come from `agentconfig.agent_feed_bytes` (shlex.split→join, tested) of the user's own `[agent] command`; fed to the durable zsh as one command line, never joined from untrusted input.
