```python
import socket
def port_free(port: int, host: str = "127.0.0.1") -> bool:
    s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    s.setsockopt(socket.SOL_SOCKET, socket.SO_REUSEADDR, 0)  # strict — no reuse
    try:
        s.bind((host, port)); return True
    except OSError:
        return False
    finally:
        s.close()
```

  ```
  for attempt in range(10):
      step = offset * (attempt + 1)              # 1000, 2000, ... (config port_offset default 1000)
      candidates = {pp: pp.published + step for pp in published}
      if all(probe(host_port) for host_port in candidates.values()):  # all candidate host ports free
          build the map; return
  raise PortAssignmentError(...)
  ```
  Concretely: on EACH attempt compute every service's candidate host port = base_published +
  (offset * (attempt+1)); probe ALL of them; if ALL free, commit; if ANY taken, bump to the next
  attempt (whole-task offset step — keeps a task's ports clustered/predictable). Cap at 10 attempts,
  then raise `PortAssignmentError`. Return an ordered dict `{service: {"base": int, "host": int,
