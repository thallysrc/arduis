     - **(a)** `self._teardown_session_terminals(task)` then `self._clear_task_state_files(task)`.
     - **(b)** `self._container_down(task)` (no-op if isolation off — its own guards).
     - **(c)** clean-gate: kick off `_conclude_clean_gate(task)` which reads
       `review.argv_status_porcelain(repo.worktree_dir)` for EACH repo (cwd=repo.worktree_dir),
