    - `@dataclass SplitNode` with `orientation: str` (values "h" = side-by-side / "v" = stacked),
      `start` and `end` (each a LeafNode | SplitNode).
    - `class LayoutModel` holding `root: LeafNode | SplitNode | None`, `focused_id: str | None`,
      a private MRU list, and a `_pre_zoom` snapshot slot. Implement:
      * `split(session_id, new_session_id, orientation="h")`: find the leaf whose session_id ==
        session_id, replace it in-place with `SplitNode(orientation, LeafNode(old), LeafNode(new))`
        (if it is the root, set root to the new SplitNode). Set focused_id = new_session_id and
        touch() it.
      * `close_leaf(session_id)`: remove the leaf; if its parent SplitNode becomes degenerate
        (one child), collapse — replace the parent with the surviving sibling subtree (handle the
        root case where the parent IS the root). Pitfall 2.
      * `visible_ids()`: in-order walk returning the session_id of every LeafNode (skip None leaves).
      * `is_visible(session_id)`: membership in visible_ids().
      * `set_leaf_session(target_id, new_id)`: swap the session_id carried by the leaf currently
        showing target_id (or the focused leaf) to new_id — the D-06 "swap into focused pane" op.
      * `zoom(session_id)`: snapshot the current root into `_pre_zoom`, set root to a single
        LeafNode(session_id). `unzoom()`: restore `_pre_zoom` (no-op if not zoomed). `is_zoomed()`:
        `_pre_zoom is not None`.
      * `preset(kind, session_ids)`: take the first N of MRU-ordered ids where N = the preset's
        cell count (grid2x2 → 4 cells; columns → len(session_ids) capped reasonably), then build a
        fresh balanced tree (grid2x2 = SplitNode("v", SplitNode("h",a,b), SplitNode("h",c,d));
        columns = a right-leaning chain of SplitNode("h",...)). Subset selection = most-recently-focused
        (D-04). Clear `_pre_zoom`.
      * `touch(session_id)`: move session_id to front of the MRU list. `mru_order()`: return the
        MRU list (most-recent first).
    - Module-level `def resolve_selection(model, session_id) -> tuple[str,str]`: return
      `("focus", session_id)` if `model.is_visible(session_id)` else `("swap", model.focused_id)` —
      pure, no GTK (D-06). This is the decision window.py acts on.
